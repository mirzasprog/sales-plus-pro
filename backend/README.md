# Retail Positions API

ASP.NET Core Web API for managing additional sales positions in supermarkets.

## Getting Started

1. Install [.NET 8 SDK](https://dotnet.microsoft.com/).
2. Navigate to `backend/src/RetailPositions.Api`.
3. Update `appsettings.json` connection string and JWT key.
4. Run database migrations (if needed) and seed sample data from `../seed/SeedData.sql`.
5. Run the API:

```bash
dotnet run
```

Swagger UI is available at `https://localhost:5001/swagger`.
